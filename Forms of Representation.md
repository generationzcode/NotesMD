##   Graph Theory
- ![[Pasted image 20250318093338.png]]
- Its a field of mathematics that studies the relationships between objects that are connected like shown above
- Edges of graphs usually have a numerical value called a weight 
- undirected graph edges have no arrows, directed have arrows
- A graph is connected if each node can reach another through some path of edges
- Order of a graph is the number of nodes, size is number of edges, degree of a vertex is number of edges connected to it
- Hamiltonian path is a path that uses every edge of a graph exactly once
- Shortest path problem minimizes the weight of the path between two notes

- The edges represent stuff like "IS A", "HAS", "PART OF", etc. between vertices (which are concepts)
