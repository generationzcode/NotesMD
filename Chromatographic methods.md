## FID

* When sample goes into the flame the flow rate is very low, to avoid the broadness of the peak.

## Chromatogram
* A plot of the detectors response as a function of the elution time is a chromatogram
* Retention time is the time between the samples injection and the maximum response
* How to know which peak is what? Use standard solutions to know the standard retention time. SO push each chemical in separately 
* Kovats retention index that standardises the retention times between all instruments. Two different instruments may have the same matrix but still give different retention times, so how do you differentiate???

## Lecture after midsems
* if a sample is a mixture of amino acids, an ion exchange resin column needs to be used to separate the consituents
* in ion exchange, the mobile phase is a buffer
* The first step of separation in ion exchange chromatography is equilibriation. The surface charge groups on the resin are connected with the mobile ions
* the exhange column that exchanges cations is called cation exchange resin - so if the attached groups to the stationary phase are negative, it's called a cation exchanger
* proteins are exchanged using a cation exchange resin since it's a cation
* The cation with more charge will be more effectively retained than one with less. It will thus elute slower
* The ion exchange column works by immobilizing the ions at certain heights in the column. To elute them, a salt gradient is used to re-equilibriate the column
* For size exclusion chromatography, the stationary phase has to be porous in nature
* kovats retention index provides a sort of fingerprint to identify substances based on their retention times
* the difference in retention factors occurs due to changes in columns so kovats retention index standardises the column basically
* MOstly in case of GC, we use volatile compunds for that we use internal standard addition method
