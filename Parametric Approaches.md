## Parametric Approaches
* There are two types of models that estimate f,parametric and non parametric
* parametric approaches basically get the problem of f down to a certain number of parameters that must be found. The model is decided from before hand and the parameters are optimized for the error
* Parametric methods must assume the form of f before fitting, this causes issues - what if f isn't actually that?
* parametric methods need less data compared to nonparametric methods for the same task
* eg. linear regressions cant be used to predict the shape of a projectile
