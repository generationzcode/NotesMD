## API gravity
- API gravity:
- $$API\space Gravity = \frac{141.5}{sp.gr\space at\space 60/60\degree F}-131.5$$
- specific gravity is the density of substance upon density of water at some temperature
- $$sp\space gr = \frac{density\space of\space substance}{density\space of\space water}$$
- crude category:
	- light crude - API gravity>38
	- medium crude - 38>API>29
	- heavy - 29>API>8.5
	- very heavy - API<8.5
## Characterization factor
- $$K = \frac{3\sqrt{TB}}{Sp.Gr \space @\space 60/60\degree F}$$
- TB = mean average boiling point in degree rankin
-  Paraffinic - K>12.1
- Intermediate - 12.1>K>11.5
- Naphthalenic - 11.5>K
