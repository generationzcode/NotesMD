> Two approaches - empirical and theoretical. Won't go into what each means because it's obvious

* In order to characterize a processing system, we need: 
* A set of fundamental dependent quantities whose values will describe the natural state of the system
* A set of equations in the fundamental quantities which will describe how the natural state of the given system changes with time
> The book then states the conservation laws as the equations - conservation of mass and energy

* The equations along with the fundamental quantities form a differential equation with time as the independent variable and the fundamental quantities as the dependent variables
* The solution of the differential equation defines the dynamic behaviour of the system
## Dead time
* The time between the inputs being changed and the effect viewed in the system is called the dead time or lag time
* Real life is never ideal and theres almost always dead time
## Input output model
* Pretty simple - a systems outputs expressed as a function of it's inputs
* $$outputs = f(inputs)$$
