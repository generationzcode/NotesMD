## Liquid vapour flow parameter
- $$F_{LV} = \frac{m_L}{m_V}\sqrt{\frac{\rho_V}{\rho_L}}$$
- $$K_1 = U_{fV,n}\sqrt{\frac{\rho_V}{\rho_L-\rho_V}}$$
- where m_L = liquid flow rate, m_V = vapour flow rate, U_{fV,n} = flooding vapour velocity and K is a constant
> Usually we know F_{LV} and we use a correlation to find K_1, from which we find the flooding velocity, which - when we divide the m_V with, we get the cross sectional area of the column

## Tray Spacing
- Typically between 450 and 750 mm
- for D>1500mm, TS>600mm recommended
- for D<450mm, TS<450mm sufficient

## Column Height
- number of trays $*$ tray spacing+TS $*$ 1.5+1500mm (above upper tray)+750mm below lower tray + 1200mm for manhole (one manhole per 8-10 trays)
