- Learning problems in machine learning are classified as either **Data Analytics** or **Adaptive Systems**
## Adaptive Systems
- Discussed this in [[Applications of AI]]
- The machine learning approach to this is using reinforcement learning
- Reinforcement learning is concerned with how "agents" take action in environments to maximize rewards
## Data Analysis
- For this, the following steps are taken:
- Start :Data harvesting, Preprocessing of data, model or theory support, feature engineering
- Training: algorithm selections, hyperparameter selection, training
- End: Post processing data, visualization
- Regression and classification are the two subsets of data analysis
- Regession predicts values of a target quantity when the target quantity is continuous
- Classification predicts the target quantity when it's discrete
