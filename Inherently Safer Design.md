## How to practice safety
* Procedural - use operating procedures, operator rules, emergency response systems, etc. This is very common but usually not enough to stop an accident from happening
* Passive. Eg-> Reactor operates at 20bar but I build a reactor rated at 100bar
* Active - manage risks using safety controls such as automatic shutdowns
* Inherent - change the process itself to avoid any events
