## Intelligent Autonomous Systems
- An intelligent autonomous system goes through 4 stages: Perceiving, thinking, acting and learning
- Perceiving consists of sensing, data collection, data aggregation and abstraction
- Thinking consists of analysis, reasoning, reflection
- Acting is done through planning the actions, configuring how it'll be done, then actually doing it
## CS vs AI
- CS is built on determinism, certainty, complete theories, quantitative backing and invariance of solution
- AI is the oppose it all the above, its adaptive with time and training
## Symbolic vs Sub-Symbolic
### Knowledge Representation
- Symbolic AI - predicate logic, sets, graphs,etc
- Sub - Neural Networks, Genetically inspired strings (?)
### Automated Reasoning
- Symbolic AI - Logic programming, Object oriented and functional programming
- Sub - propagation of values in neural nets, genetic algorithms
### Machine learning
- Symbolic - Inductive and abductive learning algorithms
- Sub - Backpropagation, genetic algorithms
